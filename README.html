NAME

SVN::Statistics

ABSTRACT

SVN::Statistics - perl module to create subversion statistics

PREREQUISITS

RRDTool and Log4Perl required

SYNOPSIS

Sample Script (please also have a look into the samples directory):

   use SVN::Statistics;

DESCRIPTION

SVN::Statistics - perl module to create subversion statistics
